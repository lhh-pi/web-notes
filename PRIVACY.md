# Privacy Policy

**Effective Date:** June 30, 2026

Web Notes does not collect, transmit, or share any user data with any third party. All data is stored locally on the user's device.

## Data Stored Locally

The extension stores the following information exclusively in the browser's built-in IndexedDB database:

| Data Category | What Is Stored | Purpose |
|---------------|----------------|---------|
| Web History | URLs, page titles, and domain names of pages where the user creates highlights | To display the list of annotated pages and allow searching by domain |
| Website Content | The text selected by the user for highlighting | To re-apply highlights when the user revisits a page |

This data never leaves the user's browser. There is no remote server, no analytics, and no tracking.

## User Control

- All data can be deleted at any time via the sidebar (delete individual pages or entire domains)
- Data can be exported as a JSON file for backup
- Uninstalling the extension removes all stored data
- The optional sync feature writes data to a local JSON file chosen by the user — the extension never uploads this file anywhere

## Third Parties

No data is shared with any third party. The extension does not use analytics, advertising, or any external services.

## Contact

For questions, open an issue at [github.com/lhh-pi/web-notes](https://github.com/lhh-pi/web-notes).

# 隐私政策

**生效日期：** 2026 年 6 月 30 日

Web Notes 不收集、不传输、不向任何第三方分享用户数据。所有数据仅存储在用户设备的浏览器本地。

## 本地存储的数据

本扩展仅将以下信息存储在浏览器内置的 IndexedDB 数据库中：

| 数据类别 | 存储内容 | 用途 |
|----------|----------|------|
| 网络记录 | 用户标注过的页面 URL、标题和域名 | 显示标注过的页面列表，支持按域名搜索 |
| 网站内容 | 用户选中高亮的原文文字 | 在用户再次访问页面时恢复高亮标记 |

这些数据不会离开用户的浏览器。没有远程服务器，没有数据分析，没有追踪。

## 用户控制

- 所有数据可随时通过侧边栏删除（单页删除或整域名删除）
- 数据可导出为 JSON 文件进行备份
- 卸载扩展会清除所有存储数据
- 可选同步功能将数据写入用户指定的本地 JSON 文件——扩展不会将此文件上传至任何地方

## 第三方

本扩展不与任何第三方分享数据。不使用分析工具、广告或任何外部服务。

## 联系方式

如有疑问，请在 [github.com/lhh-pi/web-notes](https://github.com/lhh-pi/web-notes) 提交 Issue。
